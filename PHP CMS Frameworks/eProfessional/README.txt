Release 0.0.5
 - 

Release 0.0.4
 - Added UserSettings functionality (custom Language & Template)
 - Significant security and functionality updates
 - Significant bug fixes
 - Added additional error return functionality
 - Added new classes Login(); Logout(); and ResultDisplay();
 - Added Version tracking functionality
 - Added multi-language support for the Time(); class

Release 0.0.3
 - Introduced new classes Article(); Category(); Encryption(); Error(); Template(); and UserSettings();
 - Added page direction functionality (ie: Detect if an article is chosen, category, page, etc.)
 - Added tracking of SQL queries
 - Added basic theme infrastructure
 - Included base database architecture

Release 0.0.2
 - Minor stability improvements to core api
 - Introduced new methods to the Time(); class

Release 0.0.1
 - First release of the eProfessional web platform
