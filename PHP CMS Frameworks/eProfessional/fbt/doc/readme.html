INSTALLATION REQUIREMENTS:
  • PHP 4.0.6 or higher (PHP >= 5.1.X highly recommended)
  • Upload access enabled in the PHP.INI or PHP Config (file_uploads set to "on")
  • Intermediate PHP and HTML knowledge

 NOTES:
  • The maximum dimensions of a Facebook profile picture are 200px wide by 600px high
  • It is HIGHLY recommended that, if you choose to use an overlay, you make your overlay image exactly 200px wide
  • For best result use a high quality overlay image with 300 DPI and maximum saved resolution

 IMPLEMENTATION:
  • Before the tagger can be invoked you must include the function file, by default called facebook_tagger.php
  • To invoke the tagger you will need to call the fbtTagger(); function
  • Function Paramaters fbtTagger( $uploadedImage, $goForIt );
  • $uploadedImage - Variable containing the uploaded image (ie: $_FILES['myUploadFieldName'] )
  • $goForIt - Boolean (true or false); True will render and display the image, false will simply check to ensure the image is a valid one

Return Values:

If $goForIt is set to true:
  • Returns true : Image has been succesfully rendered
  • Returns false : An error has occured
If $goForIt is set to false:
  • Returns true : Image is present and the proper format
  • Returns false : Image is not a valid resource
Both Conditions:
  • Returns $$fbtTextReturn (user set in function config) : Text reflecting the success or failure of the function
  • Returns $$fbtImageReturn (user set in function config) : Returns local URL of the image if image generation was a success
